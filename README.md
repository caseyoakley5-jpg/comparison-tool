# Comparison Tool

This repository contains the standalone `index.html` file for the LinkTheMissing Case Comparison Tool.

- Fully static (no build process)
- Designed to deploy on Netlify
- Uses Airtable API via environment variables

## Deployment

1. Upload this folder to a GitHub repository.
2. Connect the repo to Netlify.
3. Set environment variable `AIRTABLE_PAT` in Netlify.
4. Deploy.
